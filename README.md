Using this:

1. export LUA_PATH=<<THIS PROJECT DIRECTORY>>

2. make sure meelua is in lua package search path and edit ~/.config/awesome/rc.lua like this:
    require("meelua")

3. restart awesome wm (maybe mod + ctrl + r)
